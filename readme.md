## AI-Powered Diabetes Risk Prediction System

This project is a machine learning web application that predicts the risk of diabetes based on a patient's medical information. It was built to understand how machine learning models can be applied to real-world healthcare problems and how they can be deployed as interactive web applications using Streamlit.

Users can enter health details such as glucose level, blood pressure, BMI, age, insulin level, and other medical parameters. The application then predicts whether the patient is at a high or low risk of diabetes and provides useful health recommendations.

## Live Demo

🔗 https://your-streamlit-app-link.streamlit.app

---

##  Features

- Predicts diabetes risk using a Random Forest Classifier
- Interactive and user-friendly Streamlit interface
- Displays model confidence
- BMI analysis
- Health indicator dashboard
- Personalized health recommendations
- Patient summary
- Downloadable prediction report

---

##  Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

---

##  Machine Learning Model

- **Algorithm:** Random Forest Classifier
- **Dataset:** PIMA Indians Diabetes Dataset
- **Accuracy:** Approximately **76%**

##  Running the Project

Clone the repository:

```bash
git clone https://github.com/Vanshikasaini2027/diabetes-risk-prediction.git
```

Move to the project folder:

```bash
cd diabetes-risk-prediction
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
